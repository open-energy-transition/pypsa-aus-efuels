# Industry data sources

## Global Chemicals Inventory (GEM)

Source:
https://globalenergymonitor.org/projects/global-chemicals-inventory/download-data/

Dataset:
Plant-level-data-Global-Chemicals-Inventory-November-2025-V1.xlsx

License:
Creative Commons Attribution 4.0 International (CC BY 4.0)

Recommended citation:
Global Energy Monitor, Global Chemicals Inventory, November 2025 (V1) release.

Notes:
- This dataset is used for plant-level location and product information.
- It has been filtered to Australia and to ammonia and methanol plants.

## Australia ammonia and methanol production dataset

File:
ammonia_methanol_production_per_plant_au.xlsx

Description:
- Plant-level production capacities collected manually from public sources.
- Used to complement GEM data with capacity information.

Notes:
- This dataset is combined with GEM data to build the final industrial demand input.
